# Ahorcado_pygame
Segundo Parcial de Programacion 1 - Cintora Alan/Barboza Matias/Irala Facundo

Instrucciones para ejecutar el juego:
1) Favor de descargar todos los archivos incluidos en el GitHub
2) Pararse sobre el archivo parcial y ejecutarlo
3) Se abrira una ventana dando la bienvienida y explicando las instrucciones, darle click a empezar para avanzar al juego.
4) El juego cuenta con un total de 6 vidas, por cada letra que no se encuentre en la palabra, se restara una vida.
5) Una vez llegado a 6 errores, finaliza el juego como perdido.
6) Si logras adivinar correctamente la palabra, antes de perder las 6 vidas, se toma como ganado el juego.
7) Una vez ganado o perdido el juego, este se cerrara automaticamente.

*El juego cuenta con un total de 10 palabras, las cuales se eligen aleatoriamente, una vez iniciado el programa.
Las letras ingresadas se van mostrando en pantalla. Al ingresar una letra existente, el juego te avisara.
El juego cuenta con un personaje interactivo, el cual se encuentra en la parte inferior, el mismo podras moverlo de izquierda a derecha, y al llegar a los bordes de la pantalla, este colisionara, sin poder seguir avanzar.*
