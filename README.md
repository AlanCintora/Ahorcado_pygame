# 🎮 Juego del Ahorcado con Pygame

**Segundo Parcial - Programación en Python**

Este proyecto consiste en el desarrollo de un juego del *Ahorcado utilizando la biblioteca Pygame, cumpliendo con los requisitos establecidos por el documento "Segundo parcial Programacion 1 - Pygame". El código está estructurado utilizando **funciones**, **listas**, **diccionarios**, y **archivos externos**, evitando el uso de programación orientada a objetos.

---

## 📌 Objetivo

El juego busca reforzar el uso de estructuras básicas de Python mediante la creación de una interfaz gráfica que permita al usuario jugar al clásico "Ahorcado", con retroalimentación visual y sonora.

---

Instrucciones para ejecutar el juego:

Favor de descargar todos los archivos incluidos en el GitHub
Pararse sobre el archivo parcial y ejecutarlo
Se abrira una ventana dando la bienvienida y explicando las instrucciones, darle click a empezar para avanzar al juego.
El juego cuenta con un total de 6 vidas, por cada letra que no se encuentre en la palabra, se restara una vida.
Una vez llegado a 6 errores, finaliza el juego como perdido.
Si logras adivinar correctamente la palabra, antes de perder las 6 vidas, se toma como ganado el juego.
Una vez ganado o perdido el juego, este se cerrara automaticamente.
El juego cuenta con un total de 10 palabras, las cuales se eligen aleatoriamente, una vez iniciado el programa. Las letras ingresadas se van mostrando en pantalla. Al ingresar una letra existente, el juego te avisara. El juego cuenta con un personaje interactivo, el cual se encuentra en la parte inferior, el mismo podras moverlo de izquierda a derecha, y al llegar a los bordes de la pantalla, este colisionara, sin poder seguir avanzar.
